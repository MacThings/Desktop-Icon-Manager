<b>Desktop Icon Manager (DIM)</b><br>
![image](https://user-images.githubusercontent.com/51520928/113028392-115a5800-9140-11eb-83b2-c887e04ec3f5.png)

DIM is an macOS application that memorizes and restores icon positions. Multiple "arrangements" can be stored. It works for both the Desktop and Finder windows in "Icon View".
It also can toggle between showing and hiding Desktop icons.
